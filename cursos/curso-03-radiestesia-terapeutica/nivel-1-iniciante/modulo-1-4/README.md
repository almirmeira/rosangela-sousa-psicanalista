## Sobre este Módulo

O módulo final do Nível 1 é dedicado à integração de tudo o que foi aprendido. Aqui você vai entender com precisão o que é e o que não é a Radiestesia Terapêutica, descobrir onde ela pode ajudar na vida real, aprender quando encaminhar para outros profissionais, revisar os pontos principais e se preparar para o Nível 2.

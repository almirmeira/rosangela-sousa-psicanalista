## Conteúdo

### O Gráfico de Órgãos e Sistemas — Estrutura e Uso

> **⚠ ATENÇÃO ESPECIAL — ÉTICA OBRIGATÓRIA:**
> Este gráfico é o que mais exige cuidado ético. Em nenhuma hipótese seus resultados devem ser apresentados como diagnóstico, avaliação clínica ou alternativa a exames médicos.

O gráfico de órgãos é um semicírculo (ou circular) com divisões para os principais sistemas do organismo. Seu uso é estritamente orientativo — para identificar onde o campo bioenergético pode precisar de atenção complementar.

**Estrutura básica — 10 divisões:**

```
Sist.    Sist.  Aparel. Sist.   Sist.   Sist.  Sist.  Sist.  Aparel.  Sist.
Nervoso  Endo-  Digest. Cardio- Respir. Repro- Mús-   Imun.  Excret.  Osteo-
         crino          vasc.          dutivo  culoesq        ório     articular
    \      |      |       |       |      |       |      |       |      /
     \_____|______|_______|_______|______|_______|______|_______|_____/
                                    •
                                (pêndulo)
```

---

### Como Usar o Gráfico de Forma Ética e Responsável

**Perguntas adequadas:**
- "Qual sistema do corpo de [nome] pode se beneficiar de atenção energética complementar?"
- "Qual área do campo bioenergético de [nome] está com menor fluxo de energia vital?"

**Perguntas INADEQUADAS (nunca fazer):**
- "Em qual órgão está a doença de [nome]?"
- "[Nome] tem problema no [órgão X]?"
- "O coração de [nome] está saudável?"

**Resultado correto após avaliação:**
> "O gráfico indica que o sistema digestivo pode se beneficiar de atenção complementar — como aromaterapia, florais, práticas de relaxamento. Isso é uma orientação energética, não um diagnóstico. Se há qualquer sintoma físico nessa região, é fundamental consultar um médico."

### Correlações Terapêuticas (Medicina Tradicional Chinesa e Tradições Holísticas)

| Sistema | Emoção Correlata (MTC) | Suporte Complementar |
|---------|----------------------|---------------------|
| Fígado/Vesícula | Raiva, frustração | Florais (Holly, Chicory), óleo de limão |
| Rins/Bexiga | Medo, trauma | Florais (Mimulus, Rock Rose), enraizamento |
| Coração/Intestino Delg. | Alegria excessiva, ansiedade | Florais (White Chestnut), meditação |
| Pulmão/Intestino Grosso | Tristeza, luto | Florais (Gentian, Star of Bethlehem) |
| Baço/Estômago | Preocupação excessiva | Florais (White Chestnut, Elm) |

> **Nota:** Estas correlações são da Medicina Tradicional Chinesa — um sistema milenar com sua própria coerência interna. Não correspondem à fisiologia ocidental moderna. Use com clareza sobre a perspectiva.

### Protocolo Integrado: Chakras + Emoções + Sistemas

Para sessões mais completas (Nível 2–3):

```
1. Gráfico de chakras → Qual chakra em foco?
2. Gráfico de porcentagem → Nível de ativação do chakra
3. Gráfico de estados emocionais → Emoção dominante
4. Gráfico de sistemas (opcional) → Sistema energeticamente menos fluido
5. Integrar os 4 dados em uma orientação coerente
6. Selecionar intervenção complementar (florais, cristais, aromaterapia)
```

# Módulo 3.1 — Mapeamento Bioenergético Avançado

> **Nível 3 — Avançado | Curso: Radiestesia Terapêutica | Carga horária: 5h**

---

## Descrição do Módulo

O mapeamento bioenergético avançado vai além da avaliação de chakras isolados. Aqui você aprende a ver o campo energético como um sistema integrado — com relações entre os centros, padrões de bloqueio crônico, heranças energéticas e interações com o ambiente.

---

## Objetivos do Módulo

- Realizar avaliação inicial bioenergética completa
- Mapear o campo energético com múltiplas dimensões (chakras, aura, meridianos energéticos)
- Identificar padrões de bloqueio crônico e suas raízes
- Integrar o mapeamento bioenergético com o plano terapêutico
